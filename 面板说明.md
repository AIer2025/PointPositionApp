# 控制面板说明

## 轴控制（每轴相同）

| 控件 | 说明 |
|------|------|
| **当前: xx.xxx** | 实时显示轴的当前位置（mm），从 PLC 读取 |
| **使能** ☑ | 勾选后轴才能运动，相当于伺服上电 |
| **◀ ▶** | 点动按钮，**按住**移动，**松开**停止。◀ 反向，▶ 正向 |
| **手动: 50** | 点动速度（mm/s），控制 ◀ ▶ 的移动快慢 |
| **自动: 100** | 绝对运动速度（mm/s），控制自动走位的快慢 |
| **目标: 200** | 绝对运动的目标位置（mm） |
| **▷ 绝对运动** | 让轴自动走到目标位置 |
| **⌂ 回零** | 轴回原点（位置归零） |

## 夹爪控制

| 控件 | 说明 |
|------|------|
| **打开** | 夹爪张开 |
| **关闭** | 夹爪夹紧 |
| **旋转** | 夹爪旋转 |
| **打开位** | 夹爪打开时的位置值 |
| **关闭位** | 夹爪关闭时的位置值 |
| **角度** | 旋转角度 |
| **关闭力矩** | 夹紧时的力矩 |
| **打开力矩** | 张开时的力矩 |

## 顶部工具栏

| 按钮 | 说明 |
|------|------|
| **连接PLC** | 连接到 PLC（模拟模式下为虚拟连接） |
| **断开PLC** | 断开 PLC 连接 |
| **急停** | 紧急停止所有轴运动 |
| **全轴回原点** | 所有轴按优先级依次回零（Z轴优先） |
| **刷新数据** | 重新从数据库加载项目和点位数据 |
| **设置** | 打开系统设置窗口（IP、端口、模拟模式等） |

## 底部操作栏

| 按钮 | 说明 |
|------|------|
| **保存点位** | 将当前各轴位置保存到选中的点位 |
| **清除点位** | 清除选中点位的坐标数据 |
| **跳转点位** | 各轴自动运动到选中点位的坐标 |
| **批量复制(行1→全部)** | 将第一行的点位数据复制到所有行 |

## 典型操作流程

1. 勾选**使能** → 伺服上电
2. 用 **◀ ▶ 点动**微调位置，或填**目标**值点**绝对运动**自动到位
3. 在左侧点位视图中选中一个点位，点击**保存点位**记录当前坐标
4. 需要复现时，选中点位后点**跳转点位**，各轴自动走到保存的位置
