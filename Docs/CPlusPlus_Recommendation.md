# C++ 学习书籍推荐

## 入门

- **《C++ Primer》(第5版)** — Stanley Lippman 等著，公认最佳入门书，覆盖 C++11，讲解系统且扎实
- **《Programming: Principles and Practice Using C++》** — Bjarne Stroustrup（C++之父）著，适合零基础

## 进阶

- **《Effective C++》** — Scott Meyers，55条实践准则，提升代码质量必读
- **《Effective Modern C++》** — Scott Meyers，针对 C++11/14 的42条建议
- **《C++ Concurrency in Action》** — Anthony Williams，多线程/并发编程专项

## 深入理解

- **《The C++ Programming Language》(第4版)** — Stroustrup，语言权威参考
- **《Inside the C++ Object Model》** — Stanley Lippman，理解对象模型和底层机制
- **《C++ Templates: The Complete Guide》(第2版)** — 模板元编程深入指南

## 实践导向

- **《STL源码剖析》** — 侯捷，中文经典，深入理解标准库实现
- **《More Effective C++》** — Scott Meyers，Effective 系列续篇

## 建议学习路线

**C++ Primer → Effective C++ → Effective Modern C++ → 按兴趣选读深入方向**
